FTP server config

Block to configure and initialize the FTP server

Description

The FTP server config block initializes the FTP server and allows the user to set the desired username and password. Using the FTP server block it is possible to connect to the target using port 21. After authentication the user can download or delete files located on the SD card. Other actions are not possible yet.

The FTP server is currently only compatible with the WinSCP FTP Client.

Remarks

  • Place the FAT32 formatted SD-card into the slot on the system before starting the software program.
  • The FTP Server config block must be present otherwise the FTP server will not work.
  • When a file is being used by the file logger block, it will not be visible in the FTP server untill logging is stopped by using the File Logger Close block.
Parameters
Username
Desired user name for authentication when logging in to the server.
Password
Desired password for authentication when logging in to the server.
